Portfolio Site

Hello and welcome to my portfolio site. I'm an aspiring Full Stack Web Developer who is excited to learn new techniques and frameworks for all things on the web. Below you will find a list of technologies and techniques used to design this webpage. In this webpage I'm using bootstrap for the grid based responsive design.

I've created Gulp Tasks for a Web server that uses live reload. Tasks to minify my Javascript and CSS files. A watch task that will run whenever any changes are made to the tasks that are to be minified. A task for responsive images that creates 3 different images for more flexibility between devices that should let the site load faster depending on what screen size the device you're viewing the site on. This site also uses Google Fonts 'Bitter' font for text and Zocial icons in the footer and used as the icon for this website.
